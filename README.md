# Game Monetization & User Behavior Analysis

## Project Overview
This project focuses on understanding how users generate revenue in games and how different user behaviors impact monetization.

---

## Analysis Focus

### 1. Monetization Analysis
- Total revenue per game  
- Average revenue per paying user  
- Revenue concentration (top users vs others)

### 2. User Behavior Insights
- Spending patterns across age groups  
- Device impact on payment behavior  
- Language-based differences in user spending  

### 3. Efficiency Metrics
- Revenue vs ad spend  
- Cost per paying user  
- Monetization efficiency across games
